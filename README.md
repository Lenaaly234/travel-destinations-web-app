# Travel Destinations Web App

A full-stack web application to browse and search travel destinations. Includes authentication, session-based access control, and a per-user “Want-to-Go” list.

## Features
- Browse destinations with dynamic pages
- Case-insensitive keyword search with validation and “not found” handling
- Authentication + session management (protected routes)
- Per-user Want-to-Go list (add/remove + persistent storage)

## Tech Stack
- Node.js, Express.js
- MongoDB 
- EJS templating
- HTML/CSS, JavaScript
